# Instrucciones:

1. Crear su entorno virtual:
   `virtualenv .nombre`
2. Ejecutarlo con:
   `source carpeta/Scripts/activate`
3. Instalar dependencias:
   `pip install -r requirements.txt`
4. Ejecutar el proyecto:
   `python manage.py runserver`
